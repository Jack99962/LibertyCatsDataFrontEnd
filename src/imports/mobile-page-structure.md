移动端页面结构草图（文字线框）

0. 全局导航

顶部：Liberty Cats + 网络状态点（实时/延迟）+ 最后更新时间。
底部 Tab：总览 市场 持仓 地址。
全局筛选：时间范围 24H / 7D / 30D（顶部吸顶）。
1. 总览页（默认首页）

模块 A：核心 KPI（2x2 卡片）
地板价（USD）
24h 成交额
24h 成交笔数
当前上架数
每个卡片显示：当前值、环比（%）、小箭头、更新时间
模块 B：地板价 vs 成交笔数（双轴趋势图）
X 轴：时间
Y1：地板价
Y2：成交笔数
交互：长按看 tooltip，切换 24H/7D/30D
模块 C：今日结论卡（自动生成）
示例：价格震荡偏强，16:00 为峰值时段，成交额较昨日 +24.4%
2. 市场页

模块 A：24H 热力矩阵（按小时）
每格显示小时成交笔数
高亮峰值小时
模块 B：交易分布散点图
X：日期/时间
Y：成交价格
标注 top1/top2/top3 与最低成交
模块 C：异常提醒区
价格波动超阈值
成交突增/骤降
点击查看详情事件
3. 持仓页

模块 A：持仓人数趋势（折线）
当前持仓人数
WoW / MoM 变化
模块 B：持仓档位分布（横向条形）
1猫党、2-3、4-10、11-50、51-100、>100
模块 C：持有时长分布（环图）
<24H、1-7D、7-30D、30D-3M、3M-1Y、>1Y
模块 D：持仓人数变化（瀑布/条形）
各档位净增减
4. 地址页

模块 A：Top30 地址列表（卡片化）
地址短码、当前持仓、占比、环比变化、标签（官方/创始人/高频）
支持排序：按持仓/变化/占比
模块 B：Top5 增持 / 减持
两列卡片对比
模块 C：收发活跃 Top10
发送榜、接收榜，支持点击地址详情
模块 D：地址详情抽屉
历史持仓曲线
最近交易记录
外链跳转（如 OKLink）
